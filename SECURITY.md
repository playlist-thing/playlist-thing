# Security Policy

## Supported Versions

`playlist-thing` does not have planned releases. The current state of the `main` branch is supported.

## Reporting a Vulnerability

Please contact `security@playlist-thing.com` with detailed information on the vulnerability, including a Proof-Of-Concept (PoC) for exploiting the vulnerability. If your vulnerability is accepted and patched, we will credit you (if you want to be credited).
